Copy n' Haste

A Quick Sketch-y Videogame

Overview

Copy n' Haste is a videogame where you must draw counterfeit pictures of famous paintings. However, you are on the clock and must complete several drawings to the best of your ability! The more accurate your drawings are, the more money you get as a reward!

This was completed for Wesleyan University's Videogame Architecture and Programming in Unity class

Role: Programmer, Asset Integration
Duration: April 2025 (2 Weeks)
Medium: Digital Video Game (made in Unity / C#)
Team Size: 3

Responsibilities

Programming

As the sole programmer, I worked in the Unity game engine using the C# language

  • Compute shaders that read image and color information

  • Run-time texture editing that let the user draw

  • Object data scripts that save vital information for each drawing
