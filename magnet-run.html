Magnet Run

2D Magnet-Platformer

Overview

Magnet Run is a 2D platformer educational game where players must use the power of magnetism to traverse the world of Magnet Run to rescue their magnet friend!

Role: Artist, Art Integration, Programmer
Duration: February 2023 - May 2023 (13 Weeks)
Medium: Digital Video Game Demo (made in Unity / C#)
Team Size: 5

A collaborative group project, Magnet Run is an educational video game meant to teach magnetism concepts to 2nd-5th grade students This was my first time working on a videogame but I was able to contribute C# code as well as art assets and integrate all those assets into a Unity project. This project was completed within 3 months and has 4 levels. We presented this demo to the local elementary school for the students to enjoy it in a festival setting. The kids loved it and it since has fueled my game developer aspirations.
