Magnet Run

2D Magnet-Platformer

Overview

Magnet Run is a 2D platformer educational game where players must use the power of magnetism to traverse the world of Magnet Run to rescue their magnet friend!

This game was made for Wesleyan University's Videogame Development class

Role: Artist, Asset Integration, Programmer
Duration: February 2023 - May 2023 (13 Weeks)
Medium: Digital Video Game Demo (made in Unity / C#)
Team Size: 5

Responsibilities

Asset Integration / Programmer

As one of the artists and lead asset integrator, I was in charge of handling the look of Magnet Run. I also assisted in programming when needed

  • Handled creationg of sprite sheets and how they are rendered in-game

  • Utilized the timeline and animator tools to create animaitons and cutscenes

  • Programmed systems that change the sprites depending on the context

  • Co-reviewed movement and magnetism scripts, which were the main game mechanics
